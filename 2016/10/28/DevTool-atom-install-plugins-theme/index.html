[ATOM(아톰)] - 설치/플러그인/테마 설정

Oct 28, 2016 in DevTools

개요

최근에 마크다운으로 문서 작성하는 것에 재미가 생기다 보니 어플리케이션에서 마크다운 관련된 기능이 충실하게 제공되느냐가 중요하다.자바스크립트,HTML,CSS의 경우 대부분 에디터들이 충실하게 지원하기 때문에 전혀 불편한 부분이 없었다.

  • ATOM
  • VScode
  • Sublime Text

윈도우 계열에서 일할 때는 주로 서브라임 텍스트를 서브 에디터로 사용하다보니 자연스럽게 단축키를 익힐 수 있었다. 근데, 맥 환경으로 넘어와서 보니 비슷한 단축키도 많았지만 대부분 키 배열이 달라지는 바람에 버릇처럼 누르던 단축키가 아닌 다른 단축키를 일일이 찾아서 사용해야 초반에는 너무 불편했다. 사실 이것 저것 에디터를 바꿔서 사용하다보니 집중하지 못했던 것도 있는거 같다.

결정적으로 마크다운 기능은 모두 지원하고 있지만, 마크다운 성격상 빈번하게 이뤄지는 작업(이미지 업로드, 테이블 생성, HTML 변환 등) 풍부한 기능을 제공하는 플러그인이 존재하는 건 아톰밖에 보이지 않았다(물론 다른 에디터에도 기능 지원이 되는게 있는거 같은데, 일단 내 기준으로는 찾질 못해서 아톰을 다시 서브 텍스트 에디터로 선택!)

설치

설치는 쉽다. ATOM 방문해서 설치 파일 / 실행파일 다운 후 실행하면 된다. 포터블 버젼도 지원하니 설치가 싫은 유저들은 그걸 다운 받아서 진행하면 되겠다.(포터블 만세!)


설치 후 [Preference]를 들어가면 아톰의 기본적인 설정/플러그인 설치,변경 등 필요한 작업을 진행할 수 있다. 기존에 비슷한 서브라임 같은 에디터를 사용해봤다면 크게 어렵지 않게 자신에게 맞는 설정을 진행할 수 있다.

에디팅 단축키 관련 참고 URL

  • atom-shortcuts-packages
    • 누군지는 몰라도 친절하게 패키지로 단축키를 편하게 볼 수 있도록 만들어 주셨다. 고마우신분들이다.
  • nwinkler-atom-keyboard-shortcuts
  • sweetme.at-atom_cheat_sheet

설치 플러그인

markdown-preview-enhanced 플러그인에서 지원해주는 마크다운 관련 기능이 풍부한게 너무 좋았다. 번거롭게 이미지 업로드하는 부분도 클라우드에 올려주고 테이블 생성이나 마크다운 토글지원도 그렇고 마크다운 관련 플러그인 중에서는 가장 짱짱맨인거 같다.

plugins description
docblockr 주석 생성을 편하게 해준다.
emmet 태그 생성 / 탭키 활용하여 쉽게 태그를 생성할 수 있다.
file-icons 사이드바 파일들의 아이콘 표시(JS,CSS,GRUNT,NPM 등)
markdown-preview-enhanced 마크다운 관련 추가 기능 제공(이미지/테이블/링크 등)
minimap 에디팅 오른편에 미니맵 제공
minimap-highlight-selected 미니맵 하이라이팅 제공

작업하다가 이것저것 설치해보고, 좋은거는 추가로 올려놓을께요 :)

테마

테마는 크게 UI Theme / Syntax Theme 로 나뉜다. UI 테마만 변경했고, 그 중에서 가장 세련미가 넘치는(?) Seti-Ui 를 선택했다.
